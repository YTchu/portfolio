我是Jo，擅長以 UI 邏輯、結構化策略與品牌視覺延伸，並運用 AI 工具加速產品落地。工作之餘，我喜歡策略型桌遊，享受在既有規則下找出最佳得分路徑的過程。這也是我的設計思維——在技術框架與商業限制中，找出體驗與美學的最佳解方。

現任國內知名數位媒體平台資深視覺設計師。擁有豐富的視覺與網頁設計開發經驗，近幾年負責大型媒體集團旗下多個垂直分眾平台的品牌視覺延展與網站 UI 設計，涵蓋時尚潮流、質感生活、房地產、分眾社群，以及選舉開票、國際運動賽事（奧運、世足）等高流量專題。早期則從事潮流服飾 Galoop 品牌設計師，培養對品牌核心視覺趨勢的敏銳度。畢業於台南應用科技大學 ─ 視覺傳達設計系。

面對 AI 技術的快速更迭，工作中透過 Vibe Coding 與 AI 協作編程，結合原有的 HTML/CSS 切版邏輯，做出兼具視覺張力與實驗性的網頁互動特效，在設計上增加發揮空間，同時也有效降低與前端團隊的溝通成本。

我的實戰經驗專注於 Web 與 RWD 的 UI 介面邏輯，比起純粹的理論研究，我更擅長從技術邊界出發，在既有框架中運用清晰的 UI 邏輯解決複雜的產品痛點，並能與 PM、工程團隊緊密協作，將產品高效率落地。
