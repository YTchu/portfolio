目前任職於聯合新聞網資深視覺設計師。擁有豐富的視覺與網頁設計開發經驗善於將介面設計、互動效果與品牌視覺延伸，並以結構化策略與設計美感打造產品的完整度與一致性。

近幾年負責聯合新聞網多個平台品牌視覺延展與網站 UI 設計，包括 udnStyle、房地產、500 輯、女子漾、寵物部落、選舉開票與奧運、世足等專題。早期則從事潮流服飾 Galoop 品牌設計師，培養對品牌核心視覺趨勢的敏銳度。畢業於台南應用科技大學 ─ 視覺傳達設計系。
